cmd-dialog

A fast, keyboard-driven command dialog and shortcuts palette for web applications.

GitHub npm Documentation

Installation

$ pnpm add cmd-dialog

Usage

import "cmd-dialog";

// Add to your HTML
<cmd-dialog placeholder="Search..." hotkey="$mod+k"></cmd-dialog>

// Set actions programmatically
const dialog = document.querySelector("cmd-dialog");
dialog.actions = [
  { title: "Home", url: "/", hotkey: "$mod+h" },
  { title: "Settings", hotkey: "$mod+,", onAction: () => true },
];
